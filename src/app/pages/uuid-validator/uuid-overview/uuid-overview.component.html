IMUUIdValidator

This is UUId validator.

This UUId validator works with any type of UUId

Validator doesn't have any additional parameters

This validator can be imported in the next way: import {{'{'}}IMUUIdValidator{{'}'}} from 'im-angular-validators'; Error interface that validator returns can be imported as well: import {{'{'}}IMUUIdErrorInterface{{'}'}} from 'im-angular-validators';
