Parameter Type Required Default value Description
config IMNumberComparatorValidatorConfigInterface Yes No default value It defines set of configurations for the validator

Configuration has next parameter values:

Parameter Type Required Default value Description
comparingOperator IMComparatorOperatorEnum Yes No default value This is comparator operator
comparingValue number Yes, if 'comparingControlKey' is not provided No default value This is value for comparing
comparingControlKey string Yes, if 'comparingValue' is not provided No default value This is key of sibling form control (must have type number)

Next comparing operators are available:

Operator Enum key Enum value Description
= equal {{comparatorOperatorEnum.equal}} Equal to
> greater {{comparatorOperatorEnum.greater}} Greater than
>= greaterEqual {{comparatorOperatorEnum.greaterEqual}} Greater than or equal to
< less {{comparatorOperatorEnum.less}} Less than
<= lessEqual {{comparatorOperatorEnum.lessEqual}} Less than or equal to

Please, take a look at the validator examples and play with different defined validator parameters.
