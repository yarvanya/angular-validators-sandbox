Here are 'number-comparator' validator examples.

We have two possibilities to use this validator.

1. Pass comparing value directly via validator configuration

2. Pass control key directly via validator configuration

It has to be a brother form control on the same form group with model type number.

1. In next example value of the form control must be equal to 18.

What is your age? (Must be equal to 18) {{equalFormControl.errors | controlErrorResolver}}

2. Here are two form controls that are required and second one must have value equal to first one.

First Age * {{equalForm.get('firstAge').errors | controlErrorResolver}} Second Age * {{equalForm.get('secondAge').errors | controlErrorResolver}} Is form valid: {{equalForm.valid}}
