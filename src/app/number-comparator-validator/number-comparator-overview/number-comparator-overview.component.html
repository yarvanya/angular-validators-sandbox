IMNumberComparatorValidator

This is number comparator validator.

Configuration parameter must be passed with next config items:

  • Comparing Operator
  • Comparing Value
  • Comparing Control Key

We have two possibilities to use this validator.

1. Pass comparing value directly via validator configuration

2. Pass control key directly via validator configuration

It must be sibling form control from the same form group with model type number.

It can be imported in next way: import {{'{'}}IMNumberComparatorValidator{{'}'}} from 'im-angular-validators'; To pass configuration parameter, you have to import it from library: import {{'{'}}IMNumberComparatorValidatorConfigInterface{{'}'}} from 'im-angular-validators'; Also, error interface that validator returns can be imported as well: import {{'{'}}IMNumberComparatorErrorInterface{{'}'}} from 'im-angular-validators';

Please, take a look at the validator APIs for more detailed information.
