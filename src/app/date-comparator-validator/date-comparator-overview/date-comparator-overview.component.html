IMDateComparatorValidator

This is date comparator validator.

Configuration parameter must be passed with next config items:

  • Comparing Operator
  • Comparing Value
  • Comparing Control Key

We have two possibilities to use this validator.

1. Pass comparing value directly via validator configuration

2. Pass control key directly via validator configuration

It must be sibling form control from the same form group with moment model.

It can be imported in next way: import {{'{'}}IMDateComparatorValidator{{'}'}} from 'im-angular-validators'; To pass configuration parameter, you can import its interface from library: import {{'{'}}IMDateComparatorValidatorConfigInterface{{'}'}} from 'im-angular-validators'; Also, error interface that validator returns can be imported as well: import {{'{'}}IMDateComparatorErrorInterface{{'}'}} from 'im-angular-validators';

Please, take a look at the validator APIs for more detailed information.
