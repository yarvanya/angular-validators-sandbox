Here are 'date-comparator' validator examples.

We have two possibilities to use this validator.

1. Pass comparing value directly via validator configuration

2. Pass control key directly via validator configuration

It has to be a brother form control on the same form group with moment model.

1. In next example value of the form control must be to current today date.

Select date {{getErrorMessage(equalFormControl)}}

2. Here are two form controls that are required and end date has to be the same as start date.

Select start date {{getErrorMessage(equalForm.get('startDate'))}} Select end date {{getErrorMessage(equalForm.get('endDate'))}} Is form valid: {{equalForm.valid}}

1. In next example selected date must be strictly greater than current today date.

Select date {{getErrorMessage(greaterFormControl)}}

2. Here are two form controls that are required and end date must be strictly greater than start date.

Select start date {{getErrorMessage(greaterForm.get('startDate'))}} Select end date {{getErrorMessage(greaterForm.get('endDate'))}} Is form valid: {{greaterForm.valid}}

1. In next example value of the form control must be greater than or equal to current today date.

Select date {{getErrorMessage(greaterEqualFormControl)}}

2. Here are two form controls that are required and end date must be greater than or equal to start date.

Select start date {{getErrorMessage(greaterEqualForm.get('startDate'))}} Select end date {{getErrorMessage(greaterEqualForm.get('endDate'))}} Is form valid: {{greaterEqualForm.valid}}

1. In next example value of the form control must be strictly less than current today date.

Select date {{getErrorMessage(lessFormControl)}}

2. Here are two form controls that are required and end date must be strictly less than start date.

Select start date {{getErrorMessage(lessForm.get('startDate'))}} Select end date {{getErrorMessage(lessForm.get('endDate'))}} Is form valid: {{lessForm.valid}}

1. In next example value of the form control must be less than or equal to current today date.

Select date {{getErrorMessage(lessEqualFormControl)}}

2. Here are two form controls that are required and end date must be less than or equal to start date.

Select start date {{getErrorMessage(lessEqualForm.get('startDate'))}} Select end date {{getErrorMessage(lessEqualForm.get('endDate'))}} Is form valid: {{lessEqualForm.valid}}
