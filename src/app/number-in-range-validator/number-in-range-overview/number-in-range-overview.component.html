IMNumberInRangeValidator

This is number in range validator.

It defines number range, where control value must be a number in this range.

Validator has two parameters: 'from' and 'to'.

Note, that 'from' parameter must be less than 'to' parameter.

Pattern: [from, value, to]

It can be imported in next way: import {{'{'}}IMNumberInRangeValidator{{'}'}} from 'angular-validators'; Also, error interface that validator returns can be imported as well: import {{'{'}}IMNumberInRangeErrorInterface{{'}'}} from 'angular-validators';

Please, take a look at the validator APIs for more detailed information.
