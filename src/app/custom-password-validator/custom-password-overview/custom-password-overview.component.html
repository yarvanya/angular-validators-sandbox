IMPasswordValidator

This is password validator.

It has one required parameter - minLength

It can be imported in next way: import {{'{'}}IMPasswordValidator{{'}'}} from 'angular-validators'; To define and pass parameters, you have to import model from library: import {{'{'}}IMPasswordValidatorConfigModel{{'}'}} from 'angular-validators'; Also, error interface that validator returns can be imported as well: import {{'{'}}IMPasswordErrorInterface{{'}'}} from 'angular-validators';

Please, take a look at the validator APIs for more detailed information.
