Here are 'greater than number' validator examples.

We have two possibilities to use this validator.

1. Pass comparing value directly via validator configuration

In next example value of the form control must be strictly greater than 18.

What is your age? (Must be bigger than 18) {{getErrorMessage(greaterThanNumberFormControl)}}

2. Pass control key directly via validator configuration

It has to be a brother form control on the same form group with model type number.

Here are two form controls that are required and second one must have value greater than first one.

First Month * {{getErrorMessage(form.get('firstMonth'))}} Second Month * {{getErrorMessage(form.get('secondMonth'))}} Is form valid: {{form.valid}}
